== Description ==

Cpteasy is a plugin that allows you to create custom post types in a simple way and allows you to toggle the visibility of the post type in the admin menu.
An example of use is to create a custom post type for a portfolio, testimonials, etc.
A code editor allows you to update single page templates for each custom post type.

Key features include:
- Create custom post types
- Toggle visibility of the post type in the admin menu
- Update single page templates for each custom post type

== Installation ==

1. Clone the repository `git clone https://github.com/misits/cpteasy`
2. Upload the plugin files to the `/wp-content/plugins/cpteasy` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the Settings->Cpteasy screen to create new custom post types

== Frequently Asked Questions ==

= How do I create a custom post type? =

1. Go to the Cpteasy menu
2. Click on the "Create new model" tab
3. Enter all fields values
4. Click on the "Create Model" button

= How do I update the single page template for a custom post type? =

1. Go to the Cpteasy menu
2. Click on the "File editor" tab
3. Select the custom post type template on the right navigation menu
4. Update the template code
5. Click on the "Save Changes" button
